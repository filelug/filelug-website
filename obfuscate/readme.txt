Javascript 混淆檔製作
====================================================================
1. 開啟網站 https://javascriptobfuscator.com
2. 點選 Online Javascript Obfuscator 
3. 選取檔案 source/device-detector.js
4. 勾選下列3項選項設定
   i.	Encode Strings
   ii.	Move Strings
   iii.	Replace Names
5. 點選 Obfuscate  
6. 複製產生的內容後, 貼回 target/device-detector.min.js
